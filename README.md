# Camera Sentry

Camera Sentry is an Android application that periodically captures photos,
detects scene changes, and uploads meaningful images to Google Drive.

Unlike traditional time-lapse applications, Camera Sentry can automatically
discard duplicate images while preserving only meaningful scene changes.

---

## Features

- Periodic automatic photo capture
- Google Drive integration
- Daily folder creation
- Optional duplicate image removal
- Adjustable image difference threshold (1~99%)
- Adjustable capture interval
- Foreground Service
- CameraX
- Material3 UI
- Jetpack Compose
